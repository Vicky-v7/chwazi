# Changelog

## 1.0.7 (2026-05-06)

2. 新增"发表到公众号贴图"入口（需微信新版本），让选人战绩沉淀为社区内容
3. 接入官方话题 #来微信做个小程序，参与平台流量扶持

1. 优化页面标题，便于微信搜索发现

1. 新增数据埋点：分享、再来一轮、完成选人、设置变更（用于完成率/传播率分析）
2. 鸿蒙系统首次进入提示双指手势可能与系统冲突，10s 自动消失
3. 修复多指（≥6）触发系统 cancel 时圆圈被误清除的 bug
4. 选人稳定时间 1.5s → 2s，与"稳住 2 秒"文案对齐
5. 圆圈渐变重做，颜色更通透鲜艳
6. 设置页样式优化

## 1.0.1 (2026-03-26)

1. 新增分享功能（转发好友 + 分享朋友圈）
2. 修复横屏导致 canvas 尺寸错位问题，锁定竖屏
3. 启用组件按需注入，优化性能
