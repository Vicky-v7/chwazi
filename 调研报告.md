# Chwazi App 调研报告

## 一、Chwazi 是什么

Chwazi（全称 Chwazi - Finger Chooser）是一款**手指随机选人**工具，由 Tenda Digital 开发，在 iOS 和 Android 双平台上架。

核心玩法非常简单：多个人同时把手指放到屏幕上，App 会随机选中其中一个人（被选中的手指会高亮），常用于"谁来买单"、"谁先开始"等场景。

- Google Play: https://play.google.com/store/apps/details?id=com.tendadigital.chwaziApp
- App Store: https://apps.apple.com/us/app/chwazi-finger-chooser/id689674978

---

## 二、核心功能清单

| 功能 | 说明 |
|------|------|
| **随机选人** | 多人同时按屏幕，随机高亮一个手指 |
| **分组模式** | 自动将人分成若干组/队伍 |
| **选多人** | 可设置选中人数（如从 6 人中选 2 人） |
| **动画效果** | 手指按下时有圆圈动画，选中时有炫酷动效 |
| **音效** | 选中时有音效反馈 |
| **无广告** | 完全免费，无广告，不采集用户数据 |

### 使用流程

1. 设置要选几个人（或分几组）
2. 所有人把手指放到屏幕上
3. App 自动检测所有手指
4. 短暂倒计时/动画后，随机高亮选中的手指
5. 未选中的手指动画消失，选中的手指保持高亮

### 典型使用场景

- 聚餐时决定谁买单
- 桌游/卡牌游戏决定谁先开始
- 班级/团队随机分组
- 任何需要公平随机选择的场合

## 三、技术可行性分析（搬运做微信小程序）

### 1. 多点触控（核心能力）— 可行

微信小程序支持 `touchstart`、`touchmove`、`touchend` 事件，事件对象中有 `e.touches` 数组，可以获取当前屏幕上所有触摸点的坐标和标识符。这是实现 Chwazi 核心功能的关键。

**限制说明：**
- 触摸点数量取决于**设备硬件**，大多数手机支持 **5~10 个触摸点**
- 这与原生 App 面临的限制完全一样，不是小程序特有的问题
- 对于"聚会选人"的场景（通常 3~6 人），完全够用

### 2. Canvas 动画 — 可行

微信小程序的 Canvas 2.0 组件支持 2D 绑定和绘图 API，可以实现：
- 手指按下时的**彩色圆圈**绘制
- 选中动画（缩放、闪烁、光环等）
- 倒计时动画效果
- 通过 `requestAnimationFrame` 实现流畅动画循环

### 3. 音效播放 — 可行

小程序提供 `wx.createInnerAudioContext()` API，可以播放选中音效。

### 4. 无需后端 — 简单

所有逻辑都在前端完成（随机数生成、触摸检测、动画渲染），**不需要服务器**，大幅降低开发和运维成本。

### 5. 分享传播 — 天然优势

小程序比原生 App 更容易在微信群中传播分享，非常适合这种社交聚会场景。

## 四、潜在挑战和注意事项

| 挑战 | 说明 | 解决方案 |
|------|------|----------|
| **触摸点上限** | 手机硬件通常限制 5~10 点 | 实际聚会场景够用，可提示用户限制 |
| **屏幕大小** | 手机屏幕有限，多人同时按比较拥挤 | 可以用平板适配，或让每人只用一根手指 |
| **Canvas 性能** | 大量动画可能卡顿 | 使用 `requestAnimationFrame` 优化渲染 |
| **知识产权** | 直接抄袭可能有法律风险 | 做差异化设计，加入自己的创意玩法 |
| **小程序审核** | 需要符合微信审核规范 | 功能合规，无风险内容 |
| **备案要求** | 2023年9月起小程序必须备案才能上架 | 提前准备备案材料 |

## 五、差异化建议（不要只是"搬运"）

1. **加入中国本土化场景**：谁买单、谁来唱歌、真心话大冒险联动
2. **增加玩法模式**：转盘模式、摇骰子模式、惩罚选择等
3. **社交增强**：支持在微信群内直接发起选人，结果分享到群
4. **主题皮肤**：节日主题（春节红包风格、万圣节等）
5. **历史记录**：记录每次选中结果，防止"作弊"争议
6. **自定义选项**：不仅仅选人，还可以自定义选项文字（如选餐厅、选电影）

## 六、市场竞品情况

目前国内已有一些类似的随机选人工具：
- **Pick Me**（App Store）：提供"指尖挑选"功能，一次最多选 3 位
- **PickTool**（在线工具）：支持转盘、滚动、九宫格等多种选人模式
- **各种抽签小程序**：主要是输入名字后随机抽取，非手指触控模式

**差异化空间**：目前微信小程序中，基于**多指触控的手指选人**产品较少，存在市场空白。

## 七、综合评估

| 维度 | 评估 |
|------|------|
| 技术可行性 | ★★★★★ 完全可行，核心技术门槛不高 |
| 开发难度 | ★★☆☆☆ 较低，纯前端，无需后端 |
| 开发周期 | 预计 **1~2 周**可完成 MVP |
| 市场前景 | ★★★★☆ 聚会/桌游场景需求大，小程序传播性强 |
| 法律风险 | 注意做差异化，避免直接照搬 UI 和品牌 |

**结论：完全可以做，而且非常适合做成小程序。** 技术上没有障碍，多点触控 + Canvas 动画 + 随机算法就能覆盖核心功能。小程序的分享传播特性反而比原生 App 更有优势。建议在核心功能基础上做差异化创新，打造自己的品牌。
